# CI/CD
- 핵심: 개발 - 빌드 - 테스트 - 베포 까지의 전 과정을 자동화 하는 것.
- CI(Continuous Integration): 여러 개발자가 작성하거나 수정한 소스를 지속적으로 통합하고 테스트하는 것.
- CD(Continuous Delivery/Deployment): 개발, 통합, 배포, 릴리즈, 테스트를 자동화하여 지속적으로 배포하는 것.
---

## CI
- CI(Continuous Integration): 여러 개발자가 작성하거나 수정한 소스를 지속적으로 통합하고 테스트하는 것.

## CD
- CD(Continuous Delivery/Deployment): 개발, 통합, 배포, 릴리즈, 테스트를 자동화하여 지속적으로 배포하는 것.
